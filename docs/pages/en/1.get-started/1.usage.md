# 🏎️ Quick Start

Install `nuxt-vite`: (nuxt >= 2.15.0 is required)

```sh
yarn add --dev nuxt-vite
# OR
npm i -D nuxt-vite
```

Add to `buildModules`:

```js
// nuxt.config
export default {
  buildModules: [
    'nuxt-vite'
  ]
}
```

That's it! Now you can enjoy super fast `nuxt dev` experience with Vite!
